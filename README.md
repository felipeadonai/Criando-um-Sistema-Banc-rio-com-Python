# 💳 Criando um Sistema Bancário com Python

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python&logoColor=white)
![Bootcamp](https://img.shields.io/badge/Santander%20Bootcamp-DIO-red?logo=google-scholar&logoColor=white)

Este projeto foi desenvolvido como parte do **Desafio 01** do **Bootcamp Santander** promovido pela **DIO**. Ele simula um sistema bancário simples, com funcionalidades de **depósito**, **saque** e **extrato**, totalmente implementado em **Python**, utilizando o terminal.

---

## 📌 Funcionalidades

- 💰 **Depósito**: adiciona saldo à conta bancária.
- 🏧 **Saque**: permite realizar saques com limite de valor e de quantidade por dia.
- 📄 **Extrato**: mostra todas as movimentações feitas.
- 🧭 **Menu interativo**: exibição simples e clara via terminal.

---

## 🎯 Objetivo do Projeto

- Praticar conceitos fundamentais da linguagem Python.
- Utilizar:
  - Variáveis e estruturas de decisão
  - Loops e controle de fluxo
  - Funções
  - Interação com o usuário via entrada de dados
